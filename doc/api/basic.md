## BASIC COMMANDS
### init          
>Initialize ipfs local configuration

### add <path>    
>Add a file to IPFS

### cat <ref>     
>Show IPFS object data

### get <ref>     
>Download IPFS objects

### ls <ref>      
>List links from an object

### refs <ref>    
>List hashes of links from an object

- ipfs refs local 
>列出本地所有的DAG
